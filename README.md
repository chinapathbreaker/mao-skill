## 🚀 快速开始

### 安装单个技能
```bash
# 安装矛盾分析法
npx skills install /path/to/mao-skill/skills/f01-矛盾分析法

# 或复制到本地skills目录
cp -r skills/f01-矛盾分析法 ~/.agents/skills/
```

### 安装全部技能
```bash
# 批量安装所有18个技能
for dir in skills/f*; do
  npx skills install "$dir"
done
```

## 🔬 方法论

本项目采用**四层蒸馏法**：

### Phase 0: 骨架理解
- 通读全书，识别核心思想脉络
- 产出10个高价值可蒸馏技能候选

### Phase 1: 五维提取
并行运行5个提取器：
- **Frameworks**: 思维框架（15个）
- **Principles**: 核心原则（24条）
- **Cases**: 经典案例（12个）
- **Counter-examples**: 方法论反例（12个）
- **Glossary**: 术语定义（20个）

### Phase 2: RIA++构造
每个技能包含6个部分：
1. **Role**: 角色定义
2. **Input**: 输入规范
3. **Action**: 执行步骤（含Checkpoint检查点）
4. **Output**: 输出格式
5. **++Examples**: 正反例对比
6. **++Rubric**: 8维度评分标准

### Phase 3: 关系网构建
使用Zettelkasten方法建立技能间关系：
- `depends_on`: 依赖关系
- `contrasts_with`: 对比关系
- `composes_with`: 组合关系

### Phase 4: Darwin进化
- **R0**: 基线评估（18个skill，平均分79.2）
- **R1**: D4检查点修复（+3.2分）
- **R2**: 弱项专项修复（+1.7分）
- **R3**: 最终调优（+0.8分）

**最终成果**: 18个skill全部≥80分，平均分84.7

## 📝 使用示例

### 示例1: 矛盾分析
```
用户: "我们公司最近业绩下滑，请用矛盾分析法诊断"

AI: [激活f01-矛盾分析法]
1. 识别所有矛盾: 内部管理/市场竞争/产品迭代/人才流失...
2. 区分主次: 主要矛盾是产品迭代滞后于市场需求
3. 分析转化: 如果解决产品问题，人才流失会自然缓解
4. 输出: 矛盾矩阵 + 解决优先级
```

### 示例2: 调查研究
```
用户: "我想进入下沉市场，但缺乏一手数据"

AI: [激活f06-调查研究七步法]
1. 开调查会（设计访谈提纲）
2. 找典型（选择3个代表性县城）
3. 定纲目（消费力/渠道/竞品/价格敏感度）
4. 亲自出马（实地走访计划）
5. 做记录（标准化数据采集表）
6. 多验证（交叉验证数据可靠性）
7. 出结论（形成可执行洞察）
```
