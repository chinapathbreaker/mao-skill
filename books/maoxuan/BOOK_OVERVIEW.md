## 步骤2 — 解释（Interpretive）

**关键术语（按作者特定用法）**：

| 术语 | 作者的用法 |
|---|---|
| 阶级分析 | 从经济地位出发划分敌友，不是心理或道德分类 |
| 群众路线 | 从群众中来，到群众中去；领导不是保姆，是引路人 |
| 实事求是 | 从实际出发，不是从本本出发；调查是认识的前提 |
| 矛盾 | 普遍存在；有主次之分，主要矛盾决定事物性质 |
| 实践 | 认识的基础；认识从实践始，再回到实践检验 |
| 统一战线 | 团结最大多数，打击最主要的敌人 |
| 游击战争 | "敌进我退，敌驻我扰，敌疲我打，敌退我追"十六字诀 |
| 持久战 | 战略上持久，战术上速决；三阶段（防御-相持-反攻） |
| 新民主主义 | 资产阶级民主革命，由无产阶级领导；不同于旧民主主义 |
| 整风 | 反对主观主义、宗派主义、党八股；用思想改造纯洁队伍 |
| 为人民服务 | 不是做官老爷，是做人民的勤务员 |
| 人民内部矛盾 | 敌我矛盾之外的各种社会矛盾，处理方式不同于对抗性矛盾 |

**核心命题**（作者最重要的15个主张）：

1. **阶级分析法**：革命的首要问题是谁是敌人谁是朋友，而判断标准是经济剥削关系，不是道德或感情
2. **没有调查没有发言权**：一切结论产生于调查的末尾，而不是它的先头
3. **实事求是**：从实际出发，理论联系实际，反对本本主义（教条主义）
4. **实践论**：认识运动的秩序——从实践到感性认识，再到理性认识，再回到实践
5. **矛盾论**：矛盾普遍存在，但主要矛盾决定事物的性质；矛盾的主要方面决定事物的性质
6. **星星之火可以燎原**：革命高潮是"站在海岸遥望海中已经看得见桅杆尖头了的一只航船"
7. **游击战十六字诀**：敌进我退，敌驻我扰，敌疲我打，敌退我追
8. **论持久战**：三阶段论——日本强但非最强，中国弱但有广阔腹地，最终胜利属于中国
9. **新民主主义论**：资产阶级民主革命的新阶段，无产阶级领导，建立各革命阶级联合专政
10. **群众路线**：一切为了群众，一切依靠群众，从群众中来，到群众中去
11. **改造我们的学习**：理论联系实际，反对教条主义；"不注重研究现状，不注重研究历史"
12. **整顿党的作风**：反对主观主义（教条主义+经验主义）、宗派主义、党八股
13. **为人民服务**：人固有一死，或重于泰山，或轻于鸿毛；为人民利益而死，就比泰山还重
14. **论人民民主专政**：总结历史经验，论述建立人民民主专政的必然性和必要性
15. **关于正确处理人民内部矛盾**：两类不同性质的社会矛盾，敌我矛盾用专政解决，人民内部矛盾用民主团结的方式解决

**论证链**：阶级分析（认识世界）→ 调查研究（实事求是）→ 矛盾分析（把握主要矛盾）→ 实践检验（认识真理）→ 群众路线（执行方法）

## 步骤3 — 批判（Critical）

**作者的时代局限**：

- **历史背景**：写于1925-1976年，其中后期（1957年后）深受冷战格局和极左思潮影响
- **生产力判断失误**：低估了社会主义条件下发展商品经济的必要性（"跑步进入共产主义"时期）
- **阶级斗争扩大化**：后期将大量人民内部矛盾误判为敌我矛盾，导致严重后果
- **对现代经济的认识不足**：在计划经济体制认识上有明显局限

**作者的立场盲点**：

- **出身背景**：出身农民，强烈的农村视角，在某些城市政策和知识分子政策上有偏差
- **革命路径依赖**：将革命战争年代的群众动员模式应用于和平建设，存在路径错用
- **对个人崇拜的警惕不足**：在制度设计上对权力集中缺乏足够制约机制思考

**未被证明的假设**：

- 认为人的思想改造可以通过大规模运动完成（整风模式在建设时期效果有限）
- 认为阶级敌人会长期存在并不断破坏，因此需要持续革命
- 认为计划经济可以高效配置资源

**如果有人要反驳这本书，最强的论点**：
"《毛选》中后期的实践表明，将革命战争时期的阶级分析方法套用于和平建设时期，将大量认识分歧当作路线斗争，将不同意见者当作阶级敌人，会造成巨大的历史代价。"
