## 新增框架A：有理有利有节

- id: f16
  title: 有理有利有节策略框架
  type: framework
  source_chapter: 卷4·关于重庆谈判（1945.10）
  source_quote: |
    "我们的方针是和平解决问题的方针，
    但蒋介石要发动内战，我们也只是做到了一个方面。"
  summary: |
    【三字原则解析】
    有理：道义上站得住脚，我方行动有正当性
      - 在谈判桌上：我的要求是合理的，不是霸道的
      - 在冲突中：我是被迫应战的，不是主动挑衅的
    有利：实际利益上有收获，不能只图虚名
      - 不是为和平而和平，要争取实质利益
      - 谈判桌上拿不到的东西，战场上去拿
    有节：保持节制，不越界，不把对手逼到墙角
      - 不提对手无法接受的条件（保留面子）
      - 不把暂时盟友当成永久敌人

    【操作流程】
    1. 开战/谈判前：先问自己"我有理吗？"（没有理的战争/谈判迟早失败）
    2. 过程中：时刻问"我有利吗？"（没有利的协议迟早被撕毁）
    3. 收手时：问"我有节吗？"（没有节会把盟友变成敌人）

    【典型错误】
    - 只讲"有理"，忽略"有利" → 空谈道义，实际吃亏
    - 只讲"有利"，忽略"有理" → 短期利益，长期道义失分
    - 有理有利但无节 → 把对手逼到绝境，引发更大反弹
  tags: [framework, 有理有利有节, 谈判策略, 政治博弈, 新增]

## 新增框架B：农村包围城市

- id: f17
  title: 农村包围城市战略框架
  type: framework
  source_chapter: 卷1·星星之火可以燎原（1930.01）+ 卷1·井冈山的斗争（1928.11）
  source_quote: |
    "中国革命运动与俄国革命运动的最大不同，
    是中国是一个帝国主义间接统治的经济落后的半殖民地。"
  summary: |
    【核心逻辑】
    敌人力量最强的地方（城市）是进攻的终点，
    不是起点。真正的起点是敌人力量最弱的地方（农村）。

    【四步战略】
    ①建立根据地：在敌人力量薄弱的农村建立稳固的据点
      - 选择标准：地势险要、物产丰富、群众基础好
      - 目标：不是一城一地，而是建立可以自给自足的基地
    
    ②发动群众：把农民组织起来，赢得支持
      - 核心手段：土地革命（满足农民最根本的利益）
      - 效果：农民支持 → 源源不断的兵源和物资
    
    ③波浪式扩展：从点到线到面，逐步扩大根据地
      - 不是一次性占领，而是稳扎稳打
      - 每建立一个新的小块根据地，就巩固一块
    
    ④最后包围城市：当农村根据地连成一片，城市自然被包围
      - 城市是经济中心，但没有农村的支持就无法维持
      - 包围到一定程度，城市守军不战自乱

    【适用场景】
    - 任何强敌环伺的竞争格局（不是正面硬刚，而是另辟蹊径）
    - 初创企业的市场进入策略（先做细分市场，再扩展）
    - 个人职业发展（先在弱竞争领域建立优势，再挑战核心领域）
  tags: [framework, 农村包围城市, 战略, 弱者优先, 新增]

## 新增框架C：又团结又斗争原则操作化

- id: f18
  title: 又团结又斗争的动态平衡框架
  type: framework
  source_chapter: 卷3·目前抗日统一战线中的策略问题（1940.03）+ 卷6·关于正确处理人民内部矛盾（1957.02）
  source_quote: |
    "以斗争求团结则团结存，以退让求团结则团结亡。"
  summary: |
    【核心原则】
    团结是目的，斗争是手段。团结为主，斗争为辅。
    但没有斗争的团结是放弃原则的妥协，最终连团结也保不住。

    【何时团结为主】
    - 同盟者立场正确，方向一致时（维持现状，扩大合作）
    - 面对共同敌人时（内部矛盾让位于外部矛盾）
    - 同盟者的利益诉求合理时（可以协商满足的部分）

    【何时斗争为主】
    - 同盟者偏离正确方向时（需要纠正，不能放任）
    - 同盟者提出不合理要求时（拒绝，但保持关系）
    - 团结的基础被破坏时（团结的前提是原则，原则被破坏就要斗争）

    【判断框架】
    问自己三个问题：
    ①团结的目标还在吗？（核心利益是否一致）
    ②斗争的边界在哪里？（不把朋友变成敌人）
    ③团结和斗争的比例对吗？（需要动态调整）

    【常见错误】
    - 只有团结没有斗争 → 尾巴主义（失去原则）
    - 只有斗争没有团结 → 关门主义（缩小阵营）
    - 斗争过早过猛 → 破坏团结基础
    - 斗争过晚过轻 → 错失纠正时机
  tags: [framework, 又团结又斗争, 统一战线, 动态平衡, 新增]

## 新增框架D：战略持久与战术速决

- id: f19
  title: 战略持久与战术速决双轨框架（增强版）
  type: framework
  source_chapter: 卷2·论持久战（1938.05）
  source_quote: |
    "战略的持久战，战役战斗的速决战。
    这是一个道理，须得反复说明。"
  summary: |
    【双轨运行的本质】
    这是两个不同维度的决策逻辑，不能混淆：
    - 战略维度：长期、全局、方向
    - 战术维度：短期、局部、行动

    【战略层：持久】
    核心原则：保持耐力，不争一城一地之得失
    - 接受过程中的挫折和曲折（这是正常的）
    - 不被短期失败击垮信心
    - 不在力量不足时发起战略决战
    - 关键是判断"相持阶段"的转折点

    【战术层：速决】
    核心原则：每个具体行动都要快速、干净
    - 不拖延战机（机会窗口有限）
    - 不恋战（打赢了就撤，不追求最大战果）
    - 不追求完美（先行动，后优化）
    - 快速复盘，快速调整

    【两轨混淆的陷阱】
    陷阱1：用战略的"持久"为战术失败找借口
      "没事，这是持久战，慢慢来" → 实际上是拖延和不作为
    陷阱2：用战术的"速决"破坏战略节奏
      打了几个胜仗就想全面反攻 → 实际上力量还不够
    陷阱3：在战略和战术之间反复横跳
      今天想持久，明天想速决 → 没有一致性

    【操作检查表】
    □ 我是在战略层面思考还是在战术层面？
    □ 战略层面的挫折，战术层面有没有快速应对？
    □ 战术层面的胜利，有没有影响战略判断？
    □ 现在是战略的哪个阶段？（防御/相持/反攻）
  tags: [framework, 战略持久, 战术速决, 双轨框架, 增强版, 新增]
