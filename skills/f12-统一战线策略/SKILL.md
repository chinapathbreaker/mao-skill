# 统一战线动态管理策略

## R — 原文 (Reading)

> "组织千千万万的民众，调动浩浩荡荡的革命军，是今天的革命向反革命进攻的需要。""以斗争求团结则团结存，以退让求团结则团结亡。"
>
> — 毛泽东，《论反对日本帝国主义的策略》（1935.12）+ 《目前抗日统一战线中的策略问题》（1940.03）

---

## I — 方法论骨架 (Interpretation)

统一战线不是"团结一切"的道德口号，而是**动态管理敌友边界的策略工具**。

策略框架四步：
1. **识别当前阶段的主要敌人**
2. **把所有能团结的力量团结起来（哪怕是暂时的）**
3. **在统一战线内部保持独立性和批评权**
4. **当主要敌人被击败后，重新评估敌友关系**

又团结又斗争的原则：
- **团结为目的**：为了扩大力量基础
- **斗争为手段**：为了防止同盟者偏离方向
- **团结为主，斗争为辅**：但不能只有团结没有斗争（失去原则），也不能只有斗争没有团结（关门主义）

## A1 — 书中的应用 (Past Application)

### 案例 1：抗日民族统一战线（1935-1945）

- **问题**：1935年日本侵华（主要矛盾变了），过去被当作次要敌人的民族资产阶级、地主阶级，现在可以团结抗日吗？
- **方法论的使用**：建立抗日民族统一战线——对民族资产阶级：团结，但保持独立性和批评权；对地主阶级：减租减息而非没收土地；对蒋介石：逼蒋抗日（西安事变）。
- **结论**：统一战线是动态的，不是固定的；团结不能没有底线。
- **结果**：统一战线维持8年，虽然有摩擦（皖南事变），但整体维持，为抗战胜利创造了政治条件。

### 案例 2：重庆谈判中的统一战线（1945）

- **问题**：抗战胜利后，蒋介石既想打内战又需要政治准备。中共怎么利用谈判争取主动？
- **方法论的使用**：亲赴重庆谈判，展示和平诚意；同时在战场上打一仗（上党战役），用战场胜利增强谈判筹码；在谈判桌上：有理有利有节。
- **结论**：谈判是政治斗争的延续，不是妥协。
- **结果**：《双十协定》签署，政治主动权始终在中共手中。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill？

1. 需要扩大阵营，想要联合多方力量
2. 面临多方博弈，不知道该联合谁
3. 盟友内部出现分歧，不知道怎么处理
4. 过去的盟友变成了障碍，不知道是否要翻脸
5. 想要推动某件事，但力量不够

### 语言信号

- "要扩大统一战线"
- "怎么联合更多人"
- "盟友太少了"
- "这个人/这个组织怎么处理"
- "要不要拉拢他"
- "皖南事变"

### 与相邻 skill 的区分

- 与 `maoxuan-enemy-friend-dynamics` 的区别：敌友判断动态是"边界怎么划"，统一战线是"边界划定后怎么管理"
- 与 `maoxuan-struggle-unite` 的区别：又团结又斗争是统一战线的核心操作原则
- 与 `maoxuan-class-analysis` 的区别：阶级分析法是识别敌友的工具，统一战线是管理敌友关系的策略

## E — 可执行步骤 (Execution)

⚠️ **确认触发**：在开始之前，确认此场景适合使用本skill（输入'继续'进入，输入'取消'退出）
> 适用场景：需要扩大阵营、联合多方力量、盟友内部出现分歧、利益相关者管理。
> 不适用：力量悬殊（直接打就行）、信任完全崩塌、基本伦理底线被触碰。

当 skill 被激活后，按以下步骤执行：

1. **识别当前主要敌人**
   - 完成标准：用矛盾分析法（f01）识别主要矛盾，明确主要敌人是谁
   - 具体问题列表：
     1. 当前最大的矛盾是什么？（影响最多资源/威胁最大的是什么）
     2. 哪个势力是这个矛盾的主要承担者？
     3. 其他势力与这个主要矛盾的关系是什么？
   - ⏸️ **检查点**：请确认主要敌人是: [你的判断]（输入'继续'或提出修正）

2. **扩大统一战线**
   - 完成标准：列出"哪些力量可以团结"，并说明团结的条件
   - ⏸️ **检查点**：请确认团结名单和条件: [名单+条件]（输入'继续'或提出修正）

3. **保持独立性和批评权**
   - 完成标准：明确"在团结中不放弃哪些底线"

4. **动态调整（定期重评）**
   - 完成标准：写下"什么情况下要重新划定敌友边界"

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **力量悬殊碾压局**：不需要统一战线，直接打就行
- **信任完全崩塌**：关系已经不可修复，强行"团结"是自我欺骗
- **道德底线被触碰**：统一战线不能超越基本的伦理底线

### 作者在书中警告的失败模式

- 关门主义（只有斗争没有团结，盟友越来越少）
- 尾巴主义（只有团结没有斗争，失去原则和独立性）
- 团结的基础被破坏后不及时斗争
