# 矛盾转化法（量变质变框架）

## R — 原文 (Reading)

> "事物发展的根本原因，不是在事物的外部而是在事物的内部，在于事物内部的矛盾性。""矛盾着的双方，依据一定的条件，向着其相反的方向转化。"
>
> — 毛泽东，《矛盾论》，1937年8月

---

## I — 方法论骨架 (Interpretation)

矛盾转化法教你看**趋势**而非**现状**——强者什么时候会变弱？弱者什么时候会变强？

核心逻辑：
1. **矛盾双方必然相互转化**：没有永恒的强者，也没有永恒的弱者
2. **强者的"强"中包含着自我瓦解的因素**（傲慢、僵化、路径依赖）
3. **弱者的"弱"中包含着成长的基因**（灵活性、危机感、创新压力）
4. **转化的条件**：量变积累到临界点

关键心法：**多数人只能看到"现在是什么样"，看不到"它正在变成什么"。** 这是静态思维和动态思维的根本区别。

## A1 — 书中的应用 (Past Application)

### 案例 1：国共力量逆转（1921-1949）

- **问题**：1921-1934年中共极弱，国民党极强。这种力量对比会永远持续吗？
- **方法论的使用**：分析矛盾双方各自的量变积累——国民党的"强"（城市+正统性）中包含着自我瓦解（腐败、经济危机）；中共的"弱"（农村+草根）中包含着成长基因（群众基础、灵活性）。
- **结论**：量变积累到1935年后，中日民族矛盾激化（催化剂），力量对比开始逆转。
- **结果**：1949年国共易位，这个判断被历史验证。

### 案例 2：论持久战中三阶段的力量转化（1938）

- **问题**：1937年抗日战争爆发，中国极弱，日本极强。如何判断胜负？
- **方法论的使用**：分析日本"强"中的弱点（资源有限、战线太长、后劲不足）；分析中国"弱"中的强点（国土广阔、人口众多、持久力强）。
- **结论**：日本力量会逐步消耗，中国力量会逐步积累，相持阶段之后会发生质变。
- **结果**：1938年的预判与1945年的实际结果高度吻合。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill？

1. 正在经历困难期，想知道什么时候会好（或者该不该放弃）
2. 看到一个"强者"，想判断它会不会走向衰落
3. 看到一个"弱者"，想判断它有没有翻盘的可能
4. 处于顺境，想知道什么时候要警惕转折
5. 看到某个变化的信号，想判断是噪音还是质变的开始

### 语言信号

- "这个对手/趋势会变弱吗"
- "什么时候能反转"
- "现在很困难，什么时候能好"
- "强者有没有弱点"
- "量变积累到哪一步了"
- "离质变还有多远"

### 与相邻 skill 的区分

- 与 `maoxuan-contradiction-analysis` 的区别：矛盾分析法是"现在是什么样"，矛盾转化法是"它正在变成什么"（是矛盾分析的第3步延伸）
- 与 `maoxuan-persistence-war` 的区别：持久战是三阶段时间框架，矛盾转化法是力量动态框架（持久战依赖矛盾转化法）
- 与 `maoxuan-spark-fire` 的区别：星星之火评估是"新生力量有没有戏"，矛盾转化法是"力量对比如何变化"

## E — 可执行步骤 (Execution)

当 skill 被激活后，按以下步骤执行：

⚠️ **确认触发**：在开始之前，确认此场景适合使用本skill（输入'继续'进入，输入'取消'退出）

1. **分析矛盾双方的力量对比（当前）**
   - 完成标准：列出矛盾双方各自的优势和劣势

⏸️ **检查点**：请确认上述力量对比分析是否反映了你观察到的实际情况（输入'继续'或提出修正）

2. **分析变化趋势（过去6个月）**
   - 完成标准：找出哪些因素在增强，哪些在减弱（各列出至少2条）

⏸️ **检查点**：请确认这些趋势变化是否符合你的实际观察（输入'继续'或提出修正）

3. **寻找转折点（量变→质变）**
   - 完成标准：说出"量变积累到什么程度会触发质变"
   - 判停条件：若找不到明显催化剂，需要更长时间观察

⏸️ **检查点**：请确认这个转折点的判断是否合理（输入'继续'或提出修正）

4. **主动创造转化条件**
   - 完成标准：回答"什么行动能加速这个转化"

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **数据严重不足**：没有足够信息判断趋势时，不要强行预判
- **纯信息查询**：只需要了解现状，不需要判断转化方向时
- **短期高频决策**：矛盾转化法适合中长期战略，短期交易不适用

### 作者在书中警告的失败模式

- 把"现在强"当成"永远强"（诺基亚、柯达的结局）
- 把"现在弱"当成"永远弱"（错过早期特斯拉、字节跳动）
- 误判量变到质变的时间（提前放弃或过度乐观）
