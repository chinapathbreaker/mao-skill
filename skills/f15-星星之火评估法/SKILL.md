# 星星之火评估法

## R — 原文 (Reading)

> "它（革命高潮）是站在海岸遥望海中已经看得见桅杆尖头了的一只航船，它是立于高山之巅远看东方已见光芒四射喷薄欲出的一轮朝日。"
>
> — 毛泽东，《星星之火，可以燎原》，1930年1月

---

## I — 方法论骨架 (Interpretation)

星星之火评估法教你判断**新生力量是否"有戏"**，核心是看本质不看表面。

四个评估维度：
1. **内在生命力**：这个力量是否自己就能存活？（不是靠外部输血）
2. **方向正确性**：它的发展方向是否具有历史必然性？
3. **积累速度**：量变是否在加速？质变的信号是否出现？
4. **外部环境**：环境条件是否有利于它的成长？

典型错误（两种都要避免）：
- **亡国论**："现在弱小"等同于"将来也弱小"
- **速胜论**："看起来有机会"等同于"马上会成功"

---

## A1 — 书中的应用 (Past Application)

### 案例 1：1930年的"红旗还能打多久"（历史原文）

- **问题**：1927-1930年大革命失败后，革命力量极其弱小。很多人问：革命还有没有希望？红旗还能打多久？
- **方法论的使用**：毛泽东分析——革命力量虽然现在弱小，但它有内在生命力（有群众基础）；方向正确（农村包围城市）；积累正在加速（根据地不断扩大）；外部条件（日本侵华）将有利于革命力量。
- **结论**：星星之火必然燎原。
- **结果**：14年后（1949年），革命取得全国胜利。1930年的判断被历史验证。

### 案例 2：早期字节跳动的判断（2012）

- **问题**：2012年，今日头条刚上线，用户量很少，传统巨头BAT统治市场。字节跳动有没有戏？
- **方法论的使用**：分析——有内在生命力（推荐算法是真正的技术创新）；方向正确（移动互联网+个性化推荐）；积累加速（用户留存数据在变好）；外部条件（智能手机普及）为它创造了环境。
- **结论**：值得押注。
- **结果**：字节跳动成为全球最有价值独角兽之一。

### 用户会在什么情境下需要这个 skill？

1. 评估一个创业公司/项目是否值得投资

3. 评估人才的发展潜力
4. 判断某个市场是否是"值得进入的蓝海"
5. 想要在别人还没看清楚时做出判断

### 语言信号

- "这个东西有没有戏"
- "现在很弱小但以后能起来吗"
- "值得投吗"

## E — 可执行步骤 (Execution)

⚠️ **确认触发**：在开始之前，确认此场景适合使用本skill（输入'继续'进入，输入'取消'退出）
> 适用场景：评估新生力量（创业公司/技术路线/人才）是否有前途、想要早期判断。
> 不适用：已成型的成熟市场/对手评估、短期投机决策。

当 skill 被激活后，按以下步骤执行：

1. **评估内在生命力**
   - 完成标准：能回答"这个力量能自己存活吗？不需要外部输血？"
   - 量化指标：
     - 单位经济模型：是否正向（收入>成本或用户价值>获客成本）
     - 留存率：次日/7日留存是否高于行业均值50%以上
     - 自增长能力：自然流量/推荐增长占比是否>30%

2. **判断方向正确性**
   - 完成标准：能回答"它的发展方向有历史必然性吗？"
   - 量化指标：
     - 政策/技术/用户行为趋势是否明确指向这个方向
     - 市场规模是否正在扩大（年增速>20%）
   - ⏸️ **检查点**：请确认方向判断——历史必然性: [高/中/低]（输入'继续'或提出修正）

3. **观察积累速度**
   - 完成标准：能回答"量变在加速吗？质变信号出现了吗？"
   - 量化指标：
     - 用户/收入增长曲线斜率是否陡峭（环比增速>15%/月）
     - 质变信号：关键指标突破（如：市场占有率>10%，或口碑传播出现网络效应）
   - ⏸️ **检查点**：请确认积累速度——量变加速: [是/否]，质变信号: [有/无]（输入'继续'或提出修正）

4. **评估外部条件**
   - 完成标准：能回答"环境对它的成长有利吗？"
   - 量化指标：
     - 政策窗口期（是否支持/补贴该方向）
     - 技术成熟度（核心技术是否已验证）
     - 用户接受度（是否已越过采用鸿沟）

5. **给出综合判断**
   - 完成标准：明确回答"值得押注/不值得/需要更多时间观察"，并说明理由
   - 评分法：对四个维度各打1-5分，总分>16分值得押注，<10分需要谨慎
   - ⏸️ **检查点**：请确认综合判断: [值得押注/不值得/需观察]（输入'继续'或提出修正）

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **已成型的成熟竞争**：不需要判断新生力量，只需要判断现有格局
- **短期投机决策**：星星之火评估是长期判断，短期波动不适用
- **纯技术评估**：需要结合市场/社会背景，不能只看技术

### 作者在书中警告的失败模式

- 亡国论（把"现在弱小"等同于"永远弱小"）
- 速胜论（把"有机会"等同于"马上成功"）
- 忽视量变积累期（对缓慢变化缺乏敏感）
