# 调查研究七步法

## R — 原文 (Reading)

> "你对于那个问题不能解决吗？那你就去调查那个问题的现状和它的历史吧！你完完全全调查明白了，你对那个问题就有解决的办法了。""迈开你的两脚，到你的工作范围的各部分各地方去走走。学个孔夫子的'每事问'。"
>
> — 毛泽东，《反对本本主义》，1930年5月

---

## I — 方法论骨架 (Interpretation)

调查研究七步法是毛选方法论中**最具体的操作流程**，适用于任何重大决策前的准备工作。

核心原则两条：
1. **调查就是解决问题**：不是"先调查完再想怎么解决"，而是"带着问题去调查，在调查中逐步找到答案"
2. **从数据中得出结论**：不是先有结论再找证据，而是让证据引导结论

七步流程：定问题→定范围→实地访问→多维数据→量化分析→提炼规律→形成方案。

## A1 — 书中的应用 (Past Application)

### 案例 1：寻乌调查（1930）

- **问题**：制定土地政策时，党内对富农/地主的界限不清晰，政策脱离实际。
- **方法论的使用**：毛泽东花10天深入寻乌县，访问各阶层（商人、地主、富农、贫农、商人、手工业者），量化分析各阶级的人口比例和土地占有情况。
- **结论**：富农不是地主，对富农过于激进会把中农吓跑，土地政策要区分不同规模。
- **结果**：制定了更精准的土地政策，成为中共调查研究传统的奠基性案例。

### 案例 2：兴国调查（1931）

- **问题**：土地革命中，"富农"和"地主"的划分标准不统一，执行混乱。
- **方法论的使用**：毛泽东精确量化统计——按占有土地数量划分阶级，精确各阶级人口比例，分析不同阶级对革命的态度。
- **结论**：富农不是地主（自己也劳动），应与地主区别对待（抽肥补瘦而非全部没收）。
- **结果**：为更精确的土地政策提供依据。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill？

1. 要进入一个新市场/领域，完全不了解情况
2. 要制定一个重要政策，现有认知可能不够
3. 产品开发方向拿不定主意
4. 团队重组/改革推进不下去
5. 面临重大决策，但缺乏第一手数据

### 语言信号

- "我得先了解一下"
- "这个决策需要数据支撑"
- "我对这个领域完全不了解"
- "先调查再决定"
- "情况太复杂了"
- "怎么开始"

### 与相邻 skill 的区分

- 与 `maoxuan-fact-seeking` 的区别：实事求是核查表是"检查决策有没有脱离实际"，调查研究七步法是"如何系统地获取实际"
- 与 `maoxuan-impatient-left` 的区别：急躁冒进是"不等调查就下结论"，调查研究七步法是系统获取信息

## E — 可执行步骤 (Execution)

当 skill 被激活后，按以下步骤执行：

1. **定问题**（10分钟）
   - 完成标准：写出"我要做的决策是什么"，以及"做这个决策需要哪些信息"

2. **定范围**（10分钟）
   - 完成标准：确定地理/阶层/时间范围

3. **实地访问**（30分钟-多天，取决于问题复杂度）
   - 完成标准：访问至少3个不同立场的人（不能只听一方面人的话）

4. **多维数据收集**（至少覆盖人口/经济/文化/政治四个维度）
   - 完成标准：收集到可量化的数据

5. **量化分析**（找关系，不是列数字）
   - 完成标准：画出"不同群体的差异在哪里"的对比表

6. **提炼规律**（从数据中发现，不是从概念中推演）
   - 完成标准：回答"哪些因素之间有因果关系"

7. **形成方案**（可执行的行动计划）
   - 完成标准：方案包含数字目标、时间表、验收标准

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **信息充足的小决策**：不需要每个小决定都做完整调查
- **紧急危机**：来不及调查就要行动，但事后要补调查复盘
- **纯理论探讨**：学术研究有另一套方法

### 作者在书中警告的失败模式

- 先有结论，后找证据（调查变成了"验证"）
- 只听一方面人的话（汇报人说什么就是什么）
- 调查完了束之高阁（报告写得漂亮，决策没用）
